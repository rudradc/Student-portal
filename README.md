# EduNexus Student Portal 🎓

EduNexus is a comprehensive, interactive student portal system designed to bridge the gap between students, educators, and parents. The application features a dynamic, role-based architecture providing tailored dashboards for different users, deeply rooted in a professional UI/UX design foundation.

### 🎨 Design & UI/UX (Figma-Driven)
- **Figma Design Workflow:** The entire user interface was meticulously wireframed and prototyped in Figma before development. This ensures accessibility, consistent design language, and a highly intuitive user experience.
- **Premium Aesthetics:** Features a modern glassmorphism aesthetic with subtle, dynamic micro-animations, vibrant gradients, and a sleek dark-mode tailored interface.
- **Responsive Layouts:** Fluid design system that scales perfectly across desktop, tablet, and mobile devices without losing functionality.

### ✨ Key Features
- **Role-Based Dashboards:** Unique, personalized interfaces for Students, Parents, and Teachers.
- **Academic Tracking:** Real-time visibility into grades, aggregate scores, and term performance analytics.
- **Interactive Tools:** Built-in MCQ quizzes, weekly competitive contests, and a dedicated AI Doubt Solver for real-time study assistance.
- **Task Management:** Track daily homework and long-term assignments with integrated timers and priority tagging.
- **Resource Hub:** Centralized access to the official CBSE syllabus, a fully-stocked study library, and video lectures.

### 🛠️ Tech Stack
- **Frontend Development:** Semantic HTML5, Vanilla JavaScript (ES6+), Modern CSS3 (CSS Variables, Flexbox/Grid).
- **Design & Prototyping:** Figma.
